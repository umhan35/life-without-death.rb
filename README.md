# Life Without Death

This is a implementation of a self-run game called "Life Without Death" in an [assignment](assignment_spec.pdf) of "Data Structures and Algorithms" course.

## Run it

`ruby main.rb`

Also run test: `ruby LifeWithoutDeathTest.rb`
